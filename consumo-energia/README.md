- Este programa tem a finalidade de calcular o consumo mensal de KWh/mês de determinado equipamento que o usuário inserir na entrada e tanto seu custo mensal (nesse calculo estamos considerando o valor fixo de R$ 0,75 por KWh)
- O programa pode ser executando rodando o código no python (baixando o arquivo .py desde que tenha o python instalado ou usando a versão 
.exe para usuarios leigos que talvez não saibam executar e/ou não tenham o python instalado)
